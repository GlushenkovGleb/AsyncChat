# homework7

### Description:
An asynchronous chat that multiple users can use.

### Start chatting:
First you need to run a server:
```
make up
```
Then you can connect to chat:
```
make client
```
Once you don't want to chat, enter:
```
:quit
```

### Create venv:
    make venv

### Run tests:
    make test

### Run linters:
    make lint

### Run formatters:
    make format
